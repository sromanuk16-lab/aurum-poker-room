# AURUM Poker Room

AURUM Poker Room is a cinematic web/PWA poker MVP built around a luxury black-and-gold video room background with a lightweight game UI overlay.

## Core rule

> The game controls the video. The video does not control the game.

## First MVP target

- AURUM room background layer
- five center community card slots
- player seats/panels
- pot and current street
- action buttons: Fold, Check, Call, Raise
- simple local demo state without real money and without multiplayer

## Run locally

```bash
python -m http.server 8080
```

Then open `http://localhost:8080`.
