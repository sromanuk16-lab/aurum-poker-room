## Main idle loop

Use the provided image as the exact first frame and main visual reference.

Create a calm seamless cinematic loop of the AURUM luxury black-and-gold private card room. Keep the camera completely locked and static: no pan, no zoom, no dolly, no shake.

Main focus: the warm wall lamps and golden accents gently pulse with a very soft, expensive ambient flicker. The light should breathe slowly and subtly, creating delicate moving reflections on the black table edge, gold trim, glossy floor, chairs, and metallic face covering.

The final frame must visually match the first frame as closely as possible so the video can loop perfectly.

8 seconds. Seamless loop. Cinematic realistic lighting. Quiet luxury atmosphere.
