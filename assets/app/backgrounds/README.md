# AURUM App Backgrounds

Put static menu backgrounds here.

Default file expected by `src/appShell/appShell.js`:

```text
assets/app/backgrounds/home-bg.jpg
```

Recommended format:

- 9:16 portrait
- 1080x1920 or higher
- dark luxury black-gold
- no baked UI text
- no cards/chips required

If the video background is missing or disabled, the app shell falls back to this image and then to CSS gradients.
