# AURUM App Videos

Put menu background videos here.

Default file expected by `src/appShell/appShell.js`:

```text
assets/app/videos/home-idle.mp4
```

Recommended:

- 8-12 seconds loop
- portrait 9:16 or safely cropped to portrait
- dark luxury black-gold atmosphere
- no baked UI text
- no gameplay actions

The app shell works without this file. If the video is missing, it uses the static background or CSS fallback.
